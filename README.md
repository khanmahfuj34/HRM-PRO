### Adding an Employee
1. Click the **"Add New Employee"** button
2. Fill in the employee details (name, email, department, salary, etc.)
3. Click **"Save"** to add to the system

### Editing Employee Information
1. Click the **"Edit"** button on an employee card
2. Modify the required fields
3. Click **"Update"** to save changes

### Viewing Payroll Details

2. View calculated salary, deductions, and net pay
3. Export or print payroll information

2. Confirm the deletion
